What is Internet?

Written by HASIB

Internet — ek kitab ki tarah

Internet ek global network hai jo computers aur devices ko connect karta hai. Ye data packets ke form me information bhejta aur receive karta hai. DNS, TCP/IP, routers, aur servers mil kar internet ko chalate hain.

Figure: Simplified network of devices (nodes) and connections (links).

Internet kaise kaam karta hai?

Jab aap browser me kisi domain ko type karte ho, aapka device pehle DNS se IP address leta hai. Phir TCP/IP se connection ban kar packets transfer hote hain. Routers packets ko best route se aapke device tak pahunchate hain.

Type of Internet

Home Broadband

Wired services (DSL, Cable, Fiber) — gharon me high-speed internet.

Mobile Internet

Cellular networks (3G/4G/5G) — portable internet, towers aur spectrum par based.

Satellite Internet

Remote areas ke liye satellites se internet. LEO satellites latency kam kar rahe hain.

Fixed Wireless

Towers se radio waves ke through homes tak internet bhejna — fiber na hone par accha option.

Fiber Optic

Light signals ka use karke high bandwidth aur low latency provide karta hai — backbone networks me use hota hai.

All coding types (HTML · CSS · JavaScript · More)

Design by HASIB

HTML

HTML (HyperText Markup Language) web pages ka structure banata hai — headings, paragraphs, links, images sab HTML se hi bante hain.

<!-- simple HTML -->
<h1>Hello</h1>
<p>Yeh paragraph hai.</p>

CSS

CSS (Cascading Style Sheets) HTML elements ko style karta hai — colors, layout, spacing, responsiveness.

/* simple CSS */
body { font-family: Arial; }
.logo { color: blue; }

JavaScript

JavaScript se web pages interactive bante hain — buttons, forms, animations, data fetching sab JS se hota hai.

// simple JS
const btn = document.getElementById('test');
btn.addEventListener('click', ()=> alert('Hello from JS'));

Backend (Node, Python etc.)

Server-side code jo data process karta hai, databases se baat karta hai, aur APIs provide karta hai.

# simple Python Flask
from flask import Flask
app = Flask(__name__)
@app.route('/')
def home():
    return 'Hello'

Databases

Data ko store karne ke liye databases use hote hain — MySQL, PostgreSQL, MongoDB jaise options common hain.

-- sample SQL
SELECT * FROM users WHERE id = 1;

APIs & Protocols

APIs (REST/GraphQL) se alag systems interact karte hain. HTTP/HTTPS common protocols hain.
